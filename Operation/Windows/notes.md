<h1 align="center"> Windows </h1>

Windows is the operating system I use for most of my development work because im too scared to switch my OS to Linux If this isn't what you're looking for, click here to go back to the [previous page](../../NOTEBOOK.md).

Windows comes pre-installed on most machines, so there's usually nothing to install. The commands below can be used in Command Prompt, PowerShell, or Windows Terminal.

| Command                    | Description                                     |
| -------------------------- | ----------------------------------------------- |
| `cd`                       | Change the current directory.                   |
| `dir`                      | List files and folders.                         |
| `mkdir folder-name`        | Create a new folder.                            |
| `rmdir folder-name`        | Remove an empty folder.                         |
| `del file.txt`             | Delete a file.                                  |
| `copy source destination`  | Copy a file.                                    |
| `move source destination`  | Move a file or folder.                          |
| `rename old-name new-name` | Rename a file or folder.                        |
| `cls`                      | Clear the terminal screen.                      |
| `echo Hello World`         | Display text in the terminal.                   |
| `type file.txt`            | Display file contents.                          |
| `tree`                     | Display the folder structure.                   |
| `where node`               | Find the location of a command.                 |
| `whoami`                   | Show the current user.                          |
| `hostname`                 | Show the computer name.                         |
| `ipconfig`                 | Display network configuration.                  |
| `ipconfig /all`            | Display detailed network information.           |
| `ping google.com`          | Test network connectivity.                      |
| `netstat -ano`             | Show active network connections.                |
| `tasklist`                 | Show running processes.                         |
| `taskkill /PID 1234 /F`    | Force stop a process.                           |
| `systeminfo`               | Display system information.                     |
| `wmic cpu get name`        | Show CPU information.                           |
| `wmic os get caption`      | Show Windows version.                           |
| `sfc /scannow`             | Scan and repair system files.                   |
| `chkdsk`                   | Check disk health.                              |
| `shutdown /s /t 0`         | Shut down the computer.                         |
| `shutdown /r /t 0`         | Restart the computer.                           |
| `set`                      | Display environment variables.                  |
| `set VARIABLE=value`       | Create an environment variable for the session. |
| `echo %PATH%`              | Display the Path variable.                      |
| `where.exe command`        | Locate an executable.                           |
